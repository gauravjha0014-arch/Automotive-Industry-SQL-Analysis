# 🚗 Exploring Trends in the Automotive Industry Using SQL

## 📌 Project Overview

This project analyzes the used car market using SQL to uncover valuable business insights related to vehicle pricing, ownership, fuel types, mileage, engine performance, and market trends.

The analysis was performed on a dataset containing more than **8,148 vehicle records**, demonstrating practical SQL skills used in real-world data analytics projects.

---

## 🎯 Project Objectives

- Analyze vehicle pricing trends
- Compare fuel type preferences
- Study ownership impact on resale value
- Evaluate mileage and engine performance
- Explore transmission trends
- Apply Advanced SQL concepts
- Generate business insights for decision making

## 💡 Business Insights

- Premium brands command higher resale values.
- Newer vehicles have significantly higher market prices.
- Automatic vehicles generally sell at higher prices.
- First-owner vehicles retain better resale value.
- Diesel vehicles offer better mileage than petrol vehicles.
- Vehicle age is one of the biggest factors affecting depreciation.
- Manual transmission vehicles dominate the used-car market.
- Fuel type and ownership history strongly influence resale prices.
